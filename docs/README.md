# Screenshots

Drop screenshots here that the main `README.md` references.

Expected files:

- `screenshot-dashboard.png` — full party view with at least 2 characters
- `screenshot-dm.png` — DM sidebar open, with a couple of rolls visible
- `screenshot-mobile.png` (optional) — mobile layout

Recommended dimensions: 1280×720 or 1600×900. PNG, compressed.

If you fork this project, feel free to replace these with screenshots from
your own table.
